# WeChat Assistant

这是一个本地 Codex 插件，用于辅助操作已经登录的个人微信 Mac 客户端。

它的定位是“桌面 UI 助手”，不是微信协议机器人，也不是聊天记录数据库读取工具。

## 能做什么

- 打开或定位微信 Mac 客户端。
- 搜索联系人或群聊。
- 读取当前聊天窗口中可见的消息文本。
- 根据你的要求起草回复。
- 在发送前展示收件人和消息内容，等待你确认。
- 在你明确确认后，通过微信输入框发送消息。

## 不做什么

- 不解密微信本地数据库。
- 不绕过微信登录、密码、验证码或二次验证。
- 不后台持续监听聊天。
- 不读取不可见的历史记录，除非你明确要求滚动当前聊天窗口。
- 不做批量骚扰、广告群发或隐蔽发送。

## 使用方式

安装插件后，可以这样向 Codex 发起请求：

```text
用微信助手读取当前聊天窗口最后几条消息。
```

```text
给张三发微信：今天会议改到下午三点。发送前给我确认。
```

```text
打开微信，找到“项目群”，帮我草拟一条回复，但先不要发送。
```

## 默认安全规则

发送消息前必须确认以下信息：

- 收件人或群聊名称
- 完整消息内容
- 是否立即发送

如果页面要求密码、验证码、支付确认、隐私授权或系统权限，插件会停止并让你本人处理。

## 依赖

该插件依赖 Codex 的本机桌面控制能力来操作微信窗口。首次使用时，macOS 可能需要给 Codex 或相关辅助程序开启“辅助功能”和“屏幕录制”权限。
